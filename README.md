# ai-ecouter
Translate words between French and English


## Overview
Just to make it clear, this isn't a game. This app has the 1,000 most popular French words. You will be shown a word in French or English. You will be asked to translate it to the other language.


## Directions
On each turn, the game randomly selects one of the 1,000 most popular French words. It displays it in either French or English. You translate to the other language. The US and French flags indicate the languages in use.

You do not need to get "the" answer for words here. Some words have distinct multiple meanings in the other langiage. You will get credit if you use any of the common translations.

For French word answers, there are accent marks on the screen. Try to use these correctly in sp.

The game keeps tracks of the answers you get Right, Close, or Wrong. Close is for small mistakes: missing accents, or a few letters off.


## Details
For this game, it uses the following website for the common French words:
https://strommeninc.com/1000-most-common-french-words-frequency-vocabulary/

There are quirks in the list there. I went for quick and easy.

This game does minimize getting repeats. It keeps track of words selected and will avoid close repeated words.
